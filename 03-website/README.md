# 03 - Website

A searchable directory of LA homeless service resources: shelters, food,
medical care, hotlines, and a few other services. Plain HTML/CSS/JS, no
dependencies.

## Run it

Open `index.html` in a browser.

## Deploy on GitHub Pages

In **Settings -> Pages**, pick "Deploy from a branch", choose `main`, and set
the folder to `/03-website`.

## A note on the data

Resources were pulled together from public info (LAHSA, 211 LA, and the orgs'
own sites). Phone numbers, addresses, and hours change, so treat this as a
starting point and verify before relying on any single entry.

To add or fix a resource, edit the `resources` array at the top of `app.js`.
